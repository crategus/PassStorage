What is it?

PassStorage is a simple program for storing passwords, keys, or any other credentials. This program is inspired by similar one called Revelation. Currently the only supported file format is format of Revelation.

Now PassStorage successfully tested on:

  • Debian GNU/Linux sid amd64
  • Ubuntu 9.10 i386
  • Windows XP 32 bit
  • Windows 7 64 bit
All binary distributions are made with SBCL compiler. Also PassStorage can be compiled with Clozure CL version 1.4 or above.

'Yet another' password storage?

Yes and no. Originally this project was started as exercise to learn Common Lisp. Absence of Revelation on Windows platform made this program usefull for those who use both desktops Gnome and Windows. Now it is mature enough to use it daily.
